Frequently Encountered Issues (FEIs)

Approver side:

1. Valid IDs do not load(previously confirmed booking) Inform the booking agent to reattach the IDs.

2. APMS does not load Check the page Latency: (Toolbox > APMS Latency Check > start ping). If the ping is over 500 constantly, inform developer. If ping is unstable, inform IT.

3. Non-english Foreign ID (Snipping Tool or [⊞]+[Shift]+[S]) Crop screenshot the ID. Paste to Google lens > translate.

4. Spreadsheet is not visible Press f11(Fullscreen) twice (revert to window mode).

5. Spreadsheet does not load Add/ remove or change existing browser profile. Restart PC. If still doesn't work, inform IT.

Booking Agent Side:

6. Dates automatically adjusted when submitted Booking agent might be overseas. Adjust the dates by 1 day.

7. QR code is not present (Toolbox > QR Code Generator > paste ref. num.). Screenshot QR code and submit to the requestor.

8. Other unexplainable issues encountered Inform the booking agent to make another booking.
